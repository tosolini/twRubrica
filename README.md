## Installation

You'll need [Node.js](https://nodejs.org) installed on your computer in order to build this app.

```bash
$ git clone https://github.com/tosolini/twRubrica
$ cd twRubrica
$ npm install
$ npm start
```
## Translation

The main language is Italian, but i provide English too.
